# 🏥 Melbourne Health Services – Secure Enterprise Network Design

## 📌 Overview

This project demonstrates the design and implementation of a secure enterprise network for **Melbourne Health Services** using **Cisco Packet Tracer**. The network connects the Headquarters (HQ) and Branch Hospital through a hierarchical architecture that ensures scalability, redundancy, and secure communication.

## 🚀 Features

* Hierarchical Network Design
* VLAN Segmentation
* VLSM Subnetting
* Inter-VLAN Routing
* OSPF Dynamic Routing
* DHCP, DNS, Web & Email Servers
* NAT Overload (PAT)
* Site-to-Site IPSec VPN
* Extended ACLs
* SSH Remote Management
* Port Security (Sticky MAC)
* Wireless Network (WLAN)
* End-to-End Connectivity Testing

## 🏢 Network Structure

### Headquarters (HQ)

* Medical Lead Operation & Consultancy Services (MLOCS)
* Medical Emergency & Reporting (MER)
* Medical Records Management (MRM)
* Information Technology (IT)
* Customer Service (CS)

### Branch Hospital

* Hospital Labs (HL)
